# Night Clock PWA

Minimal nightstand clock for Android — large red digits on black background, landscape mode, brightness control.

## Setup (GitHub Pages)

```bash
# 1. Create repo on GitHub called "nightclock" (public)

# 2. Clone and push
git clone https://github.com/SEU-USERNAME/nightclock.git
cp index.html nightclock/
cd nightclock
git add .
git commit -m "Night Clock PWA"
git push

# 3. Enable GitHub Pages
#    Go to: Settings → Pages → Source: "main" branch → Save
#    Wait 1-2 min, URL will be: https://SEU-USERNAME.github.io/nightclock/

# 4. Open URL in Chrome Android → 3 dots → "Add to Home screen"
```
